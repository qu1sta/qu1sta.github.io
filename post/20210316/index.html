ボックスを表示するまで

16 Mar 2021, 17:49

環境

  • Unity 2020.3.0f1 Personal
  • ※：Unity 2019とAndroid 11の場合は設定がいろいろと増えるので注意

パッケージのインストール

Window > Package Manager よりPackage Manager Windowを表示 Packages: Unity Registoryに変更して以下をInstall

  • AR Foundation 4.1.5
  • ARCore XR Plugin 4.1.5

コンポーネント調整

  • Main Camera削除
  • XR -> AR Session, AR Session Originを追加

平面表示

  • XR -> AR Default Planeを追加してPrefab化する。
  • Hierarchyから AR Default Planeを削除
  • AR Session OriginのAdd ComponentからAR Plane Managerを追加。
  • Plane ManagerのPlane Prefabに上記のPrefabを追加

シーン名の変更

  • SampleSceneを別名保存でMainSceneにする。
  • SampleSceneを削除

動作確認

  • Build Settingsを開く
  • Scenes In buildにMainSceneをドラッグアンドドロップ
  • PlatformをAndroidに変更
  • Player Settingsを開く
  • Vulkan削除、Minimum APIを7.0にする。
  • XR Plug-in ManagementのARCoreにチェックを入れる

Rayの追加

  • AR Session OriginにAR Raycast Managerを追加
  • ARManager.csを作成
  • AR Session OriginのComponentにDrag&Drop
    • ※ファイル名とクラス名が違うと「The script don’t inherit a native class that can manage a script」エラーが発生。
    • 実行順序は次の通り: https://docs.unity3d.com/ja/2020.2/Manual/ExecutionOrder.html
  • CubeのPrefabを作って「Raycast Prefab」にDrag&Drop
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.XR.ARFoundation;
using UnityEngine.XR.ARSubsystems;

public class ARManager : MonoBehaviour
{
    // 生成するオブジェクト
    [SerializeField] GameObject objectPrefab;
    private ARRaycastManager raycastManager;
    private List<ARRaycastHit> hits = new List<ARRaycastHit>();
    

    // Start is called before the first frame update
    void Start()
    {
        raycastManager = GetComponent<ARRaycastManager>();
    }

    // Update is called once per frame
    void Update()
    {
        if(Input.touchCount > 0)
        {
            Touch touch = Input.GetTouch(0);
            if(touch.phase != TouchPhase.Ended)
            {
                return;
            }
            if(raycastManager.Raycast(touch.position,hits, TrackableType.All))
            {
                Instantiate(objectPrefab, hits[0].pose.position, hits[0].pose.rotation);
            }
        }
    }
}

Debugの設定

  • Package Managerから以下をインストール
  • Android Logcat 1.2.1
  • Window > Analysis > Android Logcat
