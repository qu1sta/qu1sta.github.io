ES2015及びReact

18 Oct 2019, 11:51

ES2015

波括弧について

  • Import文の波括弧なしはexport defaultされているもの、ありは通常のimport
  • 代入文は分割代入。同名のプロパティ値を代入する。

React

  • ReactはViewのみを担当
  • コンポーネントを組み合わせて画面を作る

props

  • データの受け渡し

state

  • 今の状態を表すオブジェクト
  • reducerはこれに変更を加える

React-redux

  • ReactとReduxをバインディングするライブラリ

Provider

  • Reactコンポーネント内でreact-reduxのconnect()関数を使えるようにする
  • ラップしたコンポーネントにStore情報を渡す

connect

  • ReduxのstoreにReactがアクセスするための関数
  • store情報を渡さないexport default connect()(TodoApp)
  • action createrのみを渡すexport default connect(null, actionCreators)(TodoApp)
  • stateを直接渡すexport default connect(state => state)(TodoApp)
  • stateの一部情報を渡すexport default connect(mapStateToProps)(TodoApp)
  • stateとactionの両方を渡すexport default connect(mapStateToProps, actionCreators)(TodoApp)
