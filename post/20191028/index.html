MCPC学習

28 Oct 2019, 22:28

ネットワーク

  • LTEは標準化の当初からCDMA等との接続も考慮されている

  • LTEのコアネットワークはEPC(evaluated Packet Core Network)と呼ばれ、データ用コアのみで構成されているAll-IP,音声呼はIMS(IP Multimedia Subsystem)でのVoIPもしくはCSフォールバック(CSFB)で提供されている。

  • IPネットワークの音声が劣化する原因

    • パケット遅延
    • パケットロス:UDPを使うのでパケット再送しない。伝送まちバッファのオーバーフロー
    • ジッタ：トラフィックが一定でないので、ネットワーク機器内での伝送待ち遅延のばらつき

  • CSMA/CA: 最小時間にランダム時間を加えて複数のUEからの一斉送信のコリジョンを低減させる。

  • CDMA 2000: CDMA2000 1X EV-DO Rev.Aの最大伝送速度は、下り3.1Mbps / 上り1.8Mbpsである。

  • アンテナダイバーシティ: 送信側・受信側に空間的に離れた複数のアンテナを設置することで複数の伝送路を形成

  • AAA (Adaptive Array Antenna): ビームフォーミングとヌルステアリングを併用。伝搬状況に応じた指向性制御や周波数特性制御を行う。

    • ビームフォーミング: 複数のアンテナの受信信号の振幅と位相に重みをつけて合成し、希望波の到来方向に指向性を向けてアンテナ利得を稼ぐ
    • ヌルステアリング: 不要波の到来方向に対し相対的に指向性を減衰させたヌルビームを合成する

  • IEEE802.16m: 802.16eがベースIMT-Advancedの一つとして採用されている。これをもとに。WiMAX Release2.0が策定されている。

  • 地上デジタルテレビ放送：UHF帯の470〜710MHzを利用している。6MHz幅のセグメントに区切り利用する。実際の放送に利用する13セグメントのうち、12セグメントを固定受信機向けに利用し、中央の1セグメントを携帯端末向けのワンセグ放送サービスに利用する。

  • 12セグメントを使用し、ハイビジョンテレビ番組を1番組、標準テレビ番組を2〜3番組放送可能である。

  • コグニティブ無線技術: コグニティブ無線技術とは、UEが周辺の電波の利用状況を自律的に監視して他のシステムに過度の干渉を与えることなく特定の周波数帯を利用する技術。複数の広い周波数帯に対応可能な無線回路技術や、異なる無線アクセス方式に対応するためのソフトウェア無線技術、電波の利用状況を適切に把握し無線リソースを管理する手法の確立、法整備等が必要である。

  • LTEのネットワーク構成

    • E-UTRANは基地局のみの単階層。CDMAと比べ簡素化されたネットワーク構成であるため伝送遅延と接続遅延の低減にやくだつ。

  • 直交復調器

    • 直交復調器では、受信信号を分岐し同期検波し互いに直交する２つのベースバンド信号として復調する。その後低域瀘波器で瀘波し、送信側に同期した位相で信号を判定して元の信号を取り出す。

  • マルチパス伝搬耐性技術

    • 品質確保にリードソロモン符号、ターボ符号、LDPC符号がある。

  • 移動体通信に不可欠な技術

    • 無線環境を用いる移動体通信では、電波伝搬特性を考慮しその干渉を防具技術

  • CDMAのセル構成

    • セルラー方式を採用するCDMAは基本的にすべてのセルで同一周波数が利用可能である。
    • 不感地域や屋内の感度を向上させるために、マクロセル、マイクロセル、ピコセルを使用したオーバレイ構成を取る場合は、それぞれの層に異なる周波数を使用する必要がある。

  • 無線アクセス方式のロードマップ

    • LTEはOFDMAを使用し効率よく4Gへの移行を目指し3.9G(世代)と呼ばれる。HSPA(HSDPA,HSUPA)はCDMA技術が採用され、W-CDMAの中期的な技術として3.5Gと呼ばれる。

  • LTEのインターワーキング技術

    • LTE導入にあたり、3Gと3.5G間でのインターワーキング技術が当初から考慮されている。 つまり、W-CDMAやCDMA2000とLTEの間でパケット通信や回線交換のハンドオーバが実現可能である。

  • IMTファミリーの移動体通信システム

    • 1.固定電話並みの高音質
    • 2.多彩なマルチメディアに対応する高速データ伝送
    • 3.国際的なローミング
    • 4.無線リソースの有効利用

  • 無線MAN

    • 固定WiMAXはIEEE802.16-2004に準拠
    • モバイルWiMAXはIEEE802.16e-2005に準拠
    • モバイルWiMAXは下り最大40Mbpsの高速データ伝送を実現する。また、最高120km/hの移動おｋ

  • OFDMのガードインターバルはどんな仕組みか？

    • 直接波と遅延波が生じて時差のある信号が重なって受信されるとシンボルの境界でシンボル干渉が発生。この発生したGI部分を覗いて複合する。

  • 着信転送において専用ノードが不要な理由は？ *

  • PAPよりCHAPが優れているところとは？

    • PAP: 単純にID/パスワードで認証。平文で流すので盗聴される危険
    • CHAP: チャレンジレスポンスなので、パスワードそのものを平文で流さない。

  • セル方式: サービスエリアをセルという単位に分割し、セルごとに基地局を配置する無線通信方式。

  • SIPサーバの構成

    • 登録サーバ:
      • ロケーションサーバへUAのIPアドレス情報を登録し、同じ管理ドメイン内の他のSIPサーバから利用できるようにする
    • リダイレクトサーバ:
      • リクエストの宛先を別の宛先に書き換える。一時的にUASが別の場所へ移動してもOK
    • プロキシサーバ:
      • UACからのリクエストをUASへ転送したりする。
    • ロケーションサーバ：
      • プロキシサーバ機能やリダイレクトサーバ機能によって利用されるロケーションサービスを提供。
